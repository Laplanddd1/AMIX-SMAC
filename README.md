# AMIX-SMAC

This repository contains the implementation of AMIX on SMAC (StarCraft Multi-Agent Challenge).

## Project Structure

- `src/`: Modified PyMARL source code.
- `notebooks/`: Colab training and visualization scripts.
- `results/`: Experimental results including figures, logs, and models.
- `docs/`: Graduation project documentation and slides.

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run experiments using the scripts in `src/`.
